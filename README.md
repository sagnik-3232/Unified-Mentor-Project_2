# 🍫 Nassau Candy Distributor — Logistics Dashboard

A live interactive analytics dashboard built with Streamlit and Plotly.

## 📊 Features

- **Overview** — KPI scorecard, lead time distribution, monthly trends
- **Regional** — Performance by region, heatmap, metrics table
- **Ship Mode** — Efficiency analysis, violin plots, paradox detection
- **State Routes** — Top 10 / Bottom 10, bubble chart, searchable table
- **Bottlenecks** — Delay analysis, cluster cards, critical route detection
- **Financials** — Revenue & profit by division, top products, sunburst chart

## 🚀 Run Locally

```bash
pip install -r requirements.txt
streamlit run dashboard.py
```

## 📁 Dataset

`Nassau_Candy_Distributor.csv` — 10,194 shipment records across 59 states, 4 regions, 542 cities.
